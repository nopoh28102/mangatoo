# دليل تثبيت منصة المانجا
# Manga Platform Installation Guide

## المتطلبات الأساسية / Prerequisites

### متطلبات النظام / System Requirements
- Python 3.11 أو أحدث / Python 3.11 or newer
- pip (مدير حزم Python) / pip (Python package manager)
- Git (للتحكم في الإصدارات) / Git (for version control)

### قواعد البيانات المدعومة / Supported Databases
- PostgreSQL (موصى به للإنتاج) / PostgreSQL (recommended for production)
- SQLite (للتطوير المحلي) / SQLite (for local development)

## خطوات التثبيت / Installation Steps

### 1. استنساخ المشروع / Clone the Project
```bash
git clone <repository-url>
cd manga-platform
```

### 2. إنشاء البيئة الافتراضية / Create Virtual Environment
```bash
# Windows
python -m venv manga_env
manga_env\Scripts\activate

# Linux/macOS
python3 -m venv manga_env
source manga_env/bin/activate
```

### 3. تثبيت المكتبات / Install Dependencies
```bash
pip install -r dependencies.txt
```

### 4. إعداد قاعدة البيانات / Database Setup

#### لـ SQLite (التطوير) / For SQLite (Development)
```bash
# سيتم إنشاء قاعدة البيانات تلقائياً عند تشغيل التطبيق
# Database will be created automatically when running the application
```

#### لـ PostgreSQL (الإنتاج) / For PostgreSQL (Production)
```sql
-- إنشاء قاعدة البيانات / Create Database
CREATE DATABASE manga_platform;
CREATE USER manga_user WITH PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE manga_platform TO manga_user;
```

### 5. إعداد متغيرات البيئة / Environment Variables
قم بإنشاء ملف `.env` في الجذر / Create `.env` file in root:
```env
# أساسي / Basic
SESSION_SECRET=your-secret-key-here
FLASK_ENV=development

# قاعدة البيانات / Database
DATABASE_URL=postgresql://manga_user:password@localhost/manga_platform
# أو للـ SQLite / Or for SQLite
# DATABASE_URL=sqlite:///manga_platform.db

# مجلدات الرفع / Upload Folders
UPLOAD_FOLDER=static/uploads
MAX_CONTENT_LENGTH=16777216

# مدفوعات (اختياري) / Payments (Optional)
STRIPE_SECRET_KEY=your_stripe_secret
PAYPAL_CLIENT_ID=your_paypal_client_id
PAYPAL_CLIENT_SECRET=your_paypal_secret

# استخراج المحتوى (اختياري) / Scraping (Optional)
USER_AGENT=MangaPlatform/1.0
```

### 6. إنشاء المجلدات المطلوبة / Create Required Directories
```bash
mkdir -p static/uploads/manga
mkdir -p static/uploads/covers
mkdir -p static/uploads/avatars
mkdir -p static/uploads/ads
```

### 7. تشغيل التطبيق / Run the Application

#### للتطوير / For Development
```bash
python main.py
```

#### للإنتاج / For Production
```bash
gunicorn --bind 0.0.0.0:5000 --reuse-port --reload main:app
```

## استكشاف الأخطاء / Troubleshooting

### مشاكل شائعة / Common Issues

#### خطأ في تثبيت المكتبات / Installation Errors
```bash
# تحديث pip / Update pip
pip install --upgrade pip

# تثبيت بناء الأدوات / Install build tools
pip install wheel setuptools
```

#### مشاكل قاعدة البيانات / Database Issues
```bash
# إعادة إنشاء الجداول / Recreate tables
python -c "from app import db; db.create_all()"
```

#### مشاكل الأذونات / Permission Issues
```bash
# إصلاح أذونات المجلدات / Fix folder permissions
chmod -R 755 static/uploads/
```
